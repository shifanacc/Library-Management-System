# Library-Management-System
A MySQL-based Library Management System to manage books,customers,employees,and transactions.Demonstrates key SQL concepts like joins,subqueries,and database design.

## Overview
This project is a simple Library Management System built using MySQL as part of my SQL learning journey. It helps manage books,customers,employees,and track book issue and return activities.

## Database Structure
The database includes:
* Branch details
* Employee records
* Book information and availability
* Customer data
* Book issue and return tracking

## What I learned?
While working on this project,I practiced:
* Creating tables with Primary & Foreign Keys
* Writing SQL queries with JOINs and subqueries  
* Using GROUP BY and aggregate functions  
* Designing relational databases with keys  

## Features
* Track available and issued books
* Manage customer records
* Analyze data using SQL queries
* Maintain relationships between tables

## Technologies Used
* MySQL
* SQL

## How to Run
* Create a database named library
* Create all tables using the provided SQL scripts and Insert the data
* Execute the queries

## Purpose
This project was created as part of a SQL Comprehensive Assessment to practice real-world database design and querying skills.
